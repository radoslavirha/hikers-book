# Development

## Prerequisites

- Node.js >= 18
- PNPm >= 8.7.0
- Angular CLI >= 16.2.1

## Installation

Install all the workspaces

```sh
pnpm install
```

## Build

Build all the workspaces

```sh
pnpm build
```

## Testing

Run tests in all the workspaces

```sh
pnpm test
```

## Eslint

Lint/format all the workspaces

```sh
pnpm lint
pnpm format
```
